# AI Manufacturing Diagram Prompt Generator

專業製造流程圖 Prompt 生成工具，專為 AI 繪圖模型（Midjourney / Flux / SD3 / DALL·E 等）設計。

## 功能

- 上傳產品照片或輸入文字描述
- 選擇商品名稱
- 多選製造流程站點（支援排序提示）
- 額外視覺元素（人員、看板、AGV路徑、安全標誌等）
- 一鍵產生規範的英文 Prompt（強制無中文）
- 複製到剪貼簿

## 使用方式

1. Clone 或下載本專案
2. 直接用瀏覽器開啟 `index.html`
3. 選擇流程 → 複製 Prompt → 貼到你慣用的 AI 繪圖平台
